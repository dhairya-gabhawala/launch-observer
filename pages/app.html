How to Use Launch Observer

        
A quick guide to capturing and reviewing analytics calls.

      

      

        
    
          
  1. 
            
    Start a new session
    
            
    Select a site, name the session, and optionally lock it to a specific tab.
    
          
    2. 
          
  2. 
            
    Configure the allowlist
    
            
    Choose popular services or add custom domains. Map custom domains to known services or name a custom service.
    
          
    3. 
          
  3. 
            
    Review requests
    
            
    Requests group by page. Open any request to inspect headers, query parameters, and payloads.
    
          
    4. 
          
  4. 
            
    Manage sessions
    
            
    Rename, delete, or pause sessions in the sidebar. Use Clear All Sessions to remove captured data.
